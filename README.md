# nextfem.github.io
NextFEM Dev

## Overview

This is the GitHub Pages site for the NextFEM organization. It displays all public repositories with their descriptions, programming languages, and statistics.

## Features

- Dynamic repository listing using GitHub API
- Displays only public repositories
- Shows repository descriptions, stars, forks, and open issues
- Responsive design
- XSS protection with proper HTML escaping

Visit the site at: https://nextfem.github.io/
